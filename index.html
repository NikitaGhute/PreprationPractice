Array Methods:

A. Mutable -
1. push() -add item at end of array
2. pop()- remove item from end of Array
3. shift()- delete item from start of array and shift remaining
4. unshift() - add item at start of array and reutrn new length of array
5. splice()-add or remove items at specific indices.
6. sort() - sort elements with alphabets or number order.
7. reverse() - reverses the order of items from original array.
8. fill()- fill array elements from and to specific index with specific new item.
(eg. arr.fill(fill new item, index from, index to))
